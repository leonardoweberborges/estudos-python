# Estudos de Python

Exercícios e anotações do meu processo de aprendizagem em Python.

## Objetivo

Consolidar fundamentos:

- variáveis
- condicionais
- laços de repetição
- funções
- resolução de problemas simples

## Metodologia

Sempre que aprender um conceito novo, adiciono:

1. explicação curta em linguagem própria
2. exercício resolvido
3. comentário explicando o raciocínio

## Observação

O foco não é criar projetos complexos agora, mas construir base sólida e raciocínio lógico consistente.
